# Create Reef Dapp

This is a template for creating a Reef Dapp.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/) (v12.0.0 or later)
- [Yarn](https://yarnpkg.com/) (v1.22.0 or later)

### Installation

1. npx create-reef-dapp

2. cd <name of directory>

3. yarn

4. yarn start

## Usage

### Start

```bash
yarn start
```

### Build

```bash
yarn build
```

### Test

```bash
yarn test
```
